Non-Linear Programming

Posted on 2022-05-30 Edited on 2022-11-19 In Summary Waline:

本文结合网络资源对非线性规划算法进行整理。

非线性规划

1 最速下降法

1.1 解决的问题

最速梯度下降法解决的问题是无约束优化问题，而所谓的无约束优化问题就是对目标函数的求解，没有任何的约束限制的优化问题，比如求 min⁡f(x),f:Rn→R\min f(x), f:R^n\rightarrow Rminf(x),f:Rn→R。

求解这类的问题可以分为两大类：最优条件法和迭代法。

  • 最优条件法是是指当函数存在解析形式，能够通过最优性条件求解出显式最优解。对于无约束最优化问题，如果f(x)在最优点x附近可微，那么x是局部极小点的必要条件为：df(x∗)df(x^*)df(x∗) 我们常常就是通过这个必要条件去求取可能的极小值点，再验证这些点是否真的是极小值点。当上式方程可以求解的时候，无约束最优化问题基本就解决了。
  • 实际中，这个方程往往难以求解。这就引出了第二大类方法：迭代法。

1.2 最速梯度下降法

1.2.1 算法流程

  4. 进行一维搜索，求 tkt_ktk​ 使得 f(xk+tkpk)=min⁡t≥0f(xk+tpk)f(x^k+t_kp^k)=\underset{t\ge 0}{\min}f(x^k+tp^k)f(xk+tk​pk)=t≥0min​f(xk+tpk)，令 xk+1=xk+tkpk,k=k+1x^{k+1}=x^k+t_kp^k,k=k+1xk+1=xk+tk​pk,k=k+1，转 2。

  • 梯度下降法的步长 ttt 是给定的

1.4 最速下降法的缺点

在一般情况下，当用最速下降法寻找极小点时，其搜索路径呈直角锯齿状，在开始几步，目标函数下降较快；但在接近极小点时，收敛速度就不太理想。特别是当目标函数的等值线为比较扁平的椭圆时，收敛就更慢了。

2 高斯牛顿法

2.1 牛顿法

2.1.1 用牛顿法求解零点

用牛顿法求解方程 f(x)=0f(x)=0f(x)=0 时，将 f(x)f(x)f(x) 在 xkx_kxk​ 做一阶展开

由上面部分做的一阶展开，可以得到

H(xk)≈F′′(xk)H(x_k)\approx F''(x_k) H(xk​)≈F′′(xk​)

F′(xk)+F′′(xk)Δxk=0F'(x_k)+F''(x_k)\Delta x_k=0 F′(xk​)+F′′(xk​)Δxk​=0

当上式和 H(xk)≻0H(x_k)\succ 0H(xk​)≻0 同时满足时，原目标函数在 xkx_kxk​ 的二阶展开的局部最小条件也被满足，即

Δxk=argmin⁡⁡ΔxF(xk)+F′(xk)Δx+12ΔxTF′′(xk)Δx\Delta x_k=\underset{\Delta x}{\operatorname{arg\min}}F(x_k)+F'(x_k)\Delta x+\frac{1}{2}\Delta x^TF''(x_k)\Delta x Δxk​=Δxargmin​F(xk​)+F′(xk​)Δx+21​ΔxTF′′(xk​)Δx

每一次迭代过程取增量为

Δxk=−H(xk)F′(xk)=−[∇2F(xk)]−1∇F(xk)\Delta x_k=-H(x_k)F'(x_k)=-[\nabla^2 F(x_k)]^{-1}\nabla F(x_k) Δxk​=−H(xk​)F′(xk​)=−[∇2F(xk​)]−1∇F(xk​)

dF(xk+Δxk)dΔxk=0\frac{\mathrm{d}F(x_k+\Delta x_k)}{\mathrm{d}\Delta x_k}=0 dΔxk​dF(xk​+Δxk​)​=0

第 k+1k+1k+1 轮迭代取

2.2 高斯牛顿法

高斯牛顿法针对最小二乘问题，采用一定的方法对牛顿法中的 Hesse 矩阵 H(xk)H(x_k)H(xk​) 进行近似，从而简化了计算量。

考虑最小二乘法问题

min⁡x∈RnF(x)=12∣∣f(x)∣∣2\underset{x\in\mathbb{R}^n}{\min}F(x)=\frac{1}{2}||f(x)||^2 x∈Rnmin​F(x)=21​∣∣f(x)∣∣2

当 xxx 在 xkx_kxk​ 处，具有增量 Δxk\Delta x_kΔxk​，优化目标函数取值为：

min⁡x∈RnF(xk+Δxk)=12∣∣f(xk+Δxk)∣∣2\underset{x\in\mathbb{R}^n}{\min}F(x_k+\Delta x_k)=\frac{1}{2}||f(x_k+\Delta x_k)||^2 x∈Rnmin​F(xk​+Δxk​)=21​∣∣f(xk​+Δxk​)∣∣2

不同于牛顿法，我们不对优化目标函数进行泰勒展开，我们对优化目标函数中的一部分，即 f(x)f(x)f(x) 进行一阶泰勒展开。展开后优化目标函数为：

F(xk+Δxk)=12∣∣f(xk)+J(xk)TΔxk∣∣2F(x_k+\Delta x_k)=\frac{1}{2}||f(x_k)+J(x_k)^T\Delta x_k||^2 F(xk​+Δxk​)=21​∣∣f(xk​)+J(xk​)TΔxk​∣∣2

注意，不同于牛顿法，高斯牛顿法的雅可比矩阵 J(xk)J(x_k)J(xk​) （列向量）是函数 f(x)f(x)f(x) 的，而不是目标函数。

式中仅 Δxk\Delta x_kΔxk​ 为变量，上式为一个以 Δxk\Delta x_kΔxk​ 为自变量的二次函数，且二次项系数为正（Hesse 矩阵正定 H(xk)≻0H(x_k)\succ 0H(xk​)≻0）的情况下可以取得最小值，在最小值点满足导数为 0，即

H(xk)=J(xk)J(xk)T, g(xk)=−J(xk)f(xk)H(x_k)=J(x_k)J(x_k)^T,\ g(x_k)=-J(x_k)f(x_k) H(xk​)=J(xk​)J(xk​)T, g(xk​)=−J(xk​)f(xk​)

因此得到高斯牛顿法的增量方程（搜索方向）

H(xk)Δxk=g(xk)H(x_k)\Delta x_k=g(x_k) H(xk​)Δxk​=g(xk​)

这里 H(xk)H(x_k)H(xk​) 是对 Hesse 矩阵的一种近似。

2.3 高斯牛顿法计算流程

  1. 给定初始值 x0x_0x0​，令 k=0k=0k=0；
  2. 对于第 kkk 次迭代，求出当前雅可比矩阵 J(xk)J(x_k)J(xk​) 和 f(xk)f(x_k)f(xk​) ；
  3. 求解增量方程 H(xk)Δxk=g(xk)H(x_k)\Delta x_k=g(x_k)H(xk​)Δxk​=g(xk​)；
  4. 若 Δxk\Delta x_kΔxk​ 足够小，则停止，输出 x∗≈xkx^*\approx x^kx∗≈xk。否则，令 xk+1=xk+Δxkx_{k+1}=x_k+\Delta x_kxk+1​=xk​+Δxk​，返回 2。

2.4 高斯牛顿法的缺点

  • 求解 Δxk\Delta x_kΔxk​ 时需要求解 (J(xk)J(xk)T)−1(J(x_k)J(x_k)^T)^{-1}(J(xk​)J(xk​)T)−1，但是 J(xk)J(xk)TJ(x_k)J(x_k)^TJ(xk​)J(xk​)T 只有半正定性，极有可能是奇异矩阵或病态矩阵，导致增量不稳定，算法无法收敛。
    • 奇异矩阵：不满秩矩阵。
    • 病态矩阵：求解方程组时，对数据较小的扰动就会导致结果较大的波动。
  • 若求得的 Δxk\Delta x_kΔxk​ 过大就会使得泰勒展开的局部近似不准确，导致不收敛。

3 L-M（列文伯格-马夸尔特）法

3.1 L-M 法思想

针对高斯牛顿法的不足，L-M 法做了两点改进：

  • 当求解增量 Δxk\Delta x_kΔxk​ 时，对其设置置信区间；
  • 在求解增量 Δxk\Delta x_kΔxk​ 时，对其近似效果进行了量化，并根据量化结果对信赖区域进行调整，再重新计算增量Δxk\Delta x_kΔxk​，直到近似效果量化结果达到阈值。

3.2 置信区间 μ\muμ 的设置及对增量方程的影响

3.2.1 置信区间

增加置信区间后，问题从无约束的最小二乘问题变为有约束的最小二乘问题：

min⁡x∈RnF(xk+Δxk)=12∣∣f(xk+Δxk∣∣2s.t.∣∣DΔxk∣∣2≤μ\underset{x\in\mathbb{R}^n}{\min}F(x_k+\Delta x_k)=\frac{1}{2}||f(x_k+\Delta x_k||^2\\ s.t.||D\Delta x_k||^2\le \mu x∈Rnmin​F(xk​+Δxk​)=21​∣∣f(xk​+Δxk​∣∣2s.t.∣∣DΔxk​∣∣2≤μ

式中 ∣∣DΔxk∣∣2≤μ||D\Delta x_k||^2\le \mu∣∣DΔxk​∣∣2≤μ 为置信区间的条件，DDD 是系数矩阵，为 nnn 维行向量，μ\muμ 为置信半径。

3.2.2 增量方程

列文伯格-马夸尔特在一定程度上解决了线性方程组系数矩阵的非奇异和病态问题，但算法收敛速度相较于高斯牛顿更慢。因此，当问题性质较好时，选择高斯牛顿方法；问题接近病态时，选择列文伯格-马夸尔特方法。

6 直线拟合——最小二乘

给定一些列 xi,yi(i=1,2,...,N)x_i,y_i(i=1,2,...,N)xi​,yi​(i=1,2,...,N)，假定它们具有线性关系，既可以用 y=kx+by=kx+by=kx+b 的方式进行拟合，任务是找到最优的 kkk 和 bbb 使得 xi,yix_i,y_ixi​,yi​ 和直线y=kx+by=kx+by=kx+b 最接近（一元回归）。此时问题就变为了求解目标规划

min⁡f=∑i=1N(yi−kxi−b)2\min f=\sum_{i=1}^N(y_i-kx_i-b)^2 minf=i=1∑N​(yi​−kxi​−b)2

使用线性代数的表达形式：

{y1=kx1+by2=kx2+b...yN=kxN+b\left\{ \begin{matrix} y_1=kx_1+b\\ y_2=kx_2+b\\ ...\\ y_N=kx_N+b \end{matrix} \right. ⎩⎪⎪⎨⎪⎪⎧​y1​=kx1​+by2​=kx2​+b...yN​=kxN​+b​

转为矩阵形式

[y1y2...yN]=[x11x21......xN1][kb]\left[ \begin{matrix} y_1\\ y_2\\ ...\\ y_N \end{matrix} \right] = \left[ \begin{matrix} x_1&1\\ x_2&1\\ ...&...\\ x_N&1 \end{matrix} \right] \left[ \begin{matrix} k\\ b \end{matrix} \right] ⎣⎢⎢⎡​y1​y2​...yN​​⎦⎥⎥⎤​=⎣⎢⎢⎡​x1​x2​...xN​​11...1​⎦⎥⎥⎤​[kb​]

令

Y=[y1y2...yN], K=[kb], X=[x11x21......xN1]Y= \left[ \begin{matrix} y_1\\ y_2\\ ...\\ y_N \end{matrix} \right],\ K= \left[ \begin{matrix} k\\ b \end{matrix} \right],\ X= \left[ \begin{matrix} x_1&1\\ x_2&1\\ ...&...\\ x_N&1 \end{matrix} \right] Y=⎣⎢⎢⎡​y1​y2​...yN​​⎦⎥⎥⎤​, K=[kb​], X=⎣⎢⎢⎡​x1​x2​...xN​​11...1​⎦⎥⎥⎤​

Y=XKY=XK Y=XK
