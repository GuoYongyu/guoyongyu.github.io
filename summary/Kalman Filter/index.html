Kalman Filter

Posted on 2022-09-28 Edited on 2022-11-19 In Summary Waline:

本文结合网络资源对经典卡尔曼滤波算法进行整理。

卡尔曼滤波器

1 算法介绍

考虑一种状态估计算法，已知两个值：

  • 前一时刻状态的估计值 X^k−1\hat{X}_{k-1}X^k−1​

  • 当前状态的测量值 ZkZ_kZk​

要求解当前时刻的估计值 X^k\hat{X}_kX^k​。

考虑到已知的两个值都带有噪声，一种思路则是结合这两个已知值，例如使用加权的方式：

X^k=αkZk+(1−αk)X^k−1\hat{X}_k=\alpha_k Z_k + (1-\alpha_k) \hat{X}_{k-1} X^k​=αk​Zk​+(1−αk​)X^k−1​

其中：

  • kkk 为离散的时间间隔，例如第 kkk 帧。
  • ZkZ_kZk​ 为测量值，通常伴随着噪声。
  • αk\alpha_kαk​ 为卡尔曼增益（Kalman Gain)，是未知量，用于将已知量加权。可以是固定值，但是如果根据当前状态变换这个增益，将会对当前状态的估算有很好的帮助。

卡尔曼滤波算法的思想就是利用上一状态以及当前状态测量去估计当前状态，但是模型会比上面更加复杂。

2 算法模型

卡尔曼滤波利用线性随机差分方程（Linear Stochastic Difference equation），根据上一个系统状态估计当前系统状态：

xk=Axk−1+Buk−1+wx_k = Ax_{k-1} + Bu_{k-1} + w xk​=Axk−1​+Buk−1​+w

使用时一般忽略 uuu 得到

xk=Axk−1+w(1)x_k = Ax_{k-1} + w \tag{1} xk​=Axk−1​+w(1)

测量值和状态值的线性关系描述为

zk=Hxk+v(2)z_k = Hx_k + v \tag{2} zk​=Hxk​+v(2)

其中：

  • k−1k-1k−1 和 kkk 标识上一状态和当前状态
  • x∈Rnx\in R^nx∈Rn 表示状态估计值
  • A∈Rn×nA\in R^{n\times n}A∈Rn×n 表示上一状态到当前状态的转换矩阵
  • u∈Rlu\in R^lu∈Rl 表示可选择的状态输入值，可以忽略
  • B∈Rn×lB\in R^{n\times l}B∈Rn×l 表示控制输入到当前状态的转换矩阵
  • w∈Rnw\in R^nw∈Rn 表示从上一状态到当前状态的过程噪声
  • z∈Rmz\in R^mz∈Rm 表示状态测量值
  • H∈Rm×nH\in R^{m\times n}H∈Rm×n 当前状态到当前测量的转换矩阵
  • v∈Rmv\in R^mv∈Rm 表示仪器产生的测量噪声
  • 式中 A,B,H,w,vA, B, H, w, vA,B,H,w,v 随着状态变化而变化，式中表示时没有添加下标，假设这些值不变

式中的噪声 www 和 vvv 是相互独立的，且是高斯白噪声，服从如下分布：

p(w)∼N(0,Q)p(v)∼N(0,R)Q=E(wwT), R=E(vvT)E(w)=0, E(v)=0(3)p(w)\sim N(0,Q)\\ p(v)\sim N(0,R)\\ Q=E(ww^T),\ R=E(vv^T)\\ E(w)=0,\ E(v)=0 \tag{3} p(w)∼N(0,Q)p(v)∼N(0,R)Q=E(wwT), R=E(vvT)E(w)=0, E(v)=0(3)

其中：

  • Q∈Rn×nQ\in R^{n\times n}Q∈Rn×n 和 R∈Rm×mR\in R^{m\times m}R∈Rm×m 分别表示噪声 www 和 vvv 的协方差矩阵，也会随着状态的变化而改变

注（高斯白噪声）：

  • 高斯白噪声（White Gaussian Noise）中的高斯是指概率分布是正态函数，而白噪声是指它的二阶矩不相关，一阶矩为常数，是指先后信号在时间上的相关性。
  • 定义：如果一个噪声，它的瞬时值服从高斯分布，而它的功率谱密度又是均匀分布的，则称它为高斯白噪声。

3 算法推导

定义：

  • x^ˉk\bar{\hat{x}}_kx^ˉk​ 表示不考虑过程噪声的情况下，仅利用过程先验知识求出的当前状态的先验状态估计。
  • x^k\hat{x}_kx^k​ 表示利用测量值 zkz_kzk​ 求出的当前状态的后验状态估计，也是最终求得的状态。卡尔曼滤波的最终估计就是求得 x^k\hat{x}_kx^k​，即对状态的最优估计。
  • zˉk\bar{z}_kzˉk​ 示忽略测量噪声根据当前先验状态得到的无噪声测量值。

由定义，在不考虑噪声的情况下，有

x^ˉk=Ax^k−1+Buk−1\bar{\hat{x}}_k = A\hat{x}_{k-1} + Bu_{k-1} x^ˉk​=Ax^k−1​+Buk−1​

不考虑 uuu，有

x^ˉk=Ax^k−1(4)\bar{\hat{x}}_k = A\hat{x}_{k-1} \tag{4} x^ˉk​=Ax^k−1​(4)

同样，忽略测量噪声有

zˉk=Hx^ˉk\bar{z}_k=H\bar{\hat{x}}_k zˉk​=Hx^ˉk​

有测量值减去无噪声由上一状态预测的测量值

z˙k=zk−zˉk=zk−Hx^ˉk\dot{z}_k = z_k - \bar{z}_k = z_k - H \bar{\hat{x}}_k z˙k​=zk​−zˉk​=zk​−Hx^ˉk​

  • z˙k\dot{z}_kz˙k​ 表示实际测量值（包含过程噪声及测量噪声等）减去无噪声测量值，表示预测的测量值和实际测量值的差异。实际得到是过程噪声和测量噪声对当前测量值的影响，也就是包含了 www 和 vvv。

现在，可以把无噪声的状态估计 x^ˉk\bar{\hat{x}}_kx^ˉk​ 和噪声变量 z˙k\dot{z}_kz˙k​ 按照一定的组合方式得到后验状态估计，卡尔曼滤波中使用了按照比例组合的方式：

x^k=x^ˉk+αkz˙k=x^ˉk+αk(zk−Hx^ˉk)(5)\hat{x}_k = \bar{\hat{x}}_k + \alpha_k \dot{z}_k = \bar{\hat{x}}_k + \alpha_k (z_k - H \bar{\hat{x}}_k) \tag{5} x^k​=x^ˉk​+αk​z˙k​=x^ˉk​+αk​(zk​−Hx^ˉk​)(5)

  • αk∈Rn×m\alpha_k\in R^{n\times m}αk​∈Rn×m 表示卡尔曼增益。

为求解 x^k\hat{x}_kx^k​，由于式中其他量已知，因此仅需要求出 αk\alpha_kαk​。先定义先验和后验误差：

eˉk=xk−x^ˉkek=xk−x^k\bar{e}_k = x_k - \bar{\hat{x}}_k\\ e_k = x_k - \hat{x}_k eˉk​=xk​−x^ˉk​ek​=xk​−x^k​

对应的误差协方差矩阵为：

Pˉk=E[eˉkeˉkT]Pk=E[ekekT]\bar{P}_k = E[\bar{e}_k \bar{e}_k^T]\\ P_k = E[e_k e_k^T] Pˉk​=E[eˉk​eˉkT​]Pk​=E[ek​ekT​]

为了使后验状态估计 x^k\hat{x}_kx^k​ 最优，目标是最小化后验状态估计的误差协方差矩阵 PkP_kPk​。

8 迭代过程

通过上面的公式，就可以计算状态估计，但是公式中一些参数未知，需要先定义：

  • uuu 一般被忽略
  • A,B,HA, B, HA,B,H 对于每个状态都是相同的，大部分情况下是单位阵
  • RRR 是仪器的噪声协方差
  • QQQ 较难获得，一般系统过程较为稳定，即认为没有过程噪声
  • zzz 所有状态的测量值都是由仪器测量得到而已知的

RRR 和 QQQ 的取值不是特别重要，虽然两个参数估计结果不准确，但实际情况中卡尔曼滤波算法也可以得到比较准确的状态估计。当然对噪声参数的精确也会对结果的准确度有帮助。

因此只需要传入基本的 x^0\hat{x}_0x^0​ 和 P0P_0P0​ 即可以开始进行迭代。需要注意的是，P0P_0P0​ 一般不取 000。
